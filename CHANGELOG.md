# CHANGELOG

所有重要的发布说明记录在此。

## [1.1] - 2025-12-24
### Added
- 新增考号认领流程：用户须在 `id-selection-screen` 中选择学号（01-48），学号用于排行榜与错题索引。
- 殿试模式增加倒计时准备页（5 秒）。
- 自定义放弃确认模态框替代浏览器 confirm，支持考试暂停/恢复（`isGamePaused`）。
- 设置界面增加“答对反馈”模式切换（沉浸 vs 极速）及高级数据管理按钮（重置刷题记忆、清空榜单、销毁错题）。
- 闪卡与复习模块交互优化（翻转动画、进度条、随机标题）。

### Changed
- 计时精度提升（以 0.01s 更新），超时处理与视觉反馈改进。
- Hard (殿试) 模式题库记忆使用 `hard_mixed` 键以支持混合题库追踪。

### Notes
- 发行渠道：班级内部分发（班级版）。
- 提示：在大规模分发前请在干净的 Windows 环境中测试安装程序并考虑代码签名以减少安全提示。
