## 📥 安装方法

### 方法 1️⃣  - 使用安装程序（推荐）

**如果你得到了 `.exe` 安装文件：**

1. 双击 `国史科举·五经通考 Setup X.X.X.exe`
2. 点击 "I Agree" 接受许可证
3. 选择安装位置（默认为 Program Files）
4. 点击 "Install" 开始安装
5. 等待安装完成
6. 安装程序会自动创建桌面快捷方式
7. 双击桌面上的快捷方式启动应用

### 方法 2️⃣  - 使用便携版（无需安装）

**如果你得到了便携版 `.exe`：**

1. 将 EXE 文件复制到任意位置（如桌面、USB、网盘）
2. 直接双击运行
3. 应用会立即启动

### 方法 3️⃣  - 使用启动脚本（开发者模式）

**如果你得到了项目源代码：**

1. 确保已安装 Node.js (https://nodejs.org/)
2. 在项目文件夹找到 `run.bat` 或 `start.bat`
3. 双击运行
4. 脚本会自动安装依赖并启动应用

---

### v1.1 新增与注意事项（重要）

本版本（v1.1，班级版）在启动与考试流程上有若干重要变化，请务必阅读：

- 考号认领：首次进入考试流程时，你需要在“考号认领”页面选择你的考号（01-48）。考号会作为排名和错题记录的索引保存到本地。若未选择考号，无法进入难度选择页面。

- 殿试倒计时：选择“殿试（难）”后会进入预备倒计时页面（5 秒），请集中注意力，倒计时结束后自动进入答题。

- 放弃确认与暂停：在答题过程中点击“放弃赶考”会弹出自定义确认模态框（替代浏览器原生 confirm），此时考试处于暂停状态；若取消放弃会恢复考试并继续计时。

- 答题计时更精确：时间以 0.01s 更新，超时会自动记录为错题并进入下一题。

- 设置项：设置中新增“答对反馈：沉浸模式 / 极速模式”切换，控制答对时的视觉特效（沉浸模式会有金光特效与更长展示时间）。

- 数据管理：设置页增加“重置刷题记忆（全题库）”、“清空所有排行榜”、“销毁所有错题本”三个危险性操作，执行前会弹出确认提示，请谨慎操作。

这些变更会影响排行榜与错题的识别（使用考号作为键），建议教师在班级发放前通知学生正确认领考号。

## 🎮 应用功能

### 📝 进京赶考 - 参加科举考试
- **乡试（易）**：分科考核，每科 4 选 1，15 秒时间限制
- **殿试（难）**：全科混考，6 选 1，5 秒时间限制
- 连对 5 题获得秀才，15 题获得举人，30 题获得进士

### 🎴 知识闪卡 - 记忆历史知识
- 各科知识点闪卡

- 点击翻转查看答案
- 进度追踪

- 人物卷：历代重要人物介绍
- 地理卷：历代疆域信息
- 文化卷：重要文化制度
- 经济卷：经济制度演变
- 官职卷：历代官职体系
- 农桑卷：农业相关知识

### 🏆 皇榜 - 查看排行榜
- 分难度（乡试/殿试）查看排名
- 显示学号、功名、积分
- 追踪班级成绩

### 📂 错题阁 - 管理错题
- 查看和分析错题
- 统计错题次数
- 帮助针对性复习

### ⚙️  游戏设置 - 个性化配置
- 调整答题反馈效果
- 管理数据（清空数据、重置进度等）

## ❓ 常见问题

### Q: 为什么首次启动比较慢？
**A:** 首次启动需要初始化应用环境。后续启动会更快。

### Q: 数据会不会丢失？
**A:** 不会。所有数据存储在本地，除非你手动删除或清空数据。

### Q: 能在多台电脑上使用吗？
**A:** 可以。每台电脑的数据是独立的。

### Q: 如何导出数据？
**A:** 目前暂不支持直接导出。数据存储在 Windows 用户文件夹中。

### Q: 应用占用多少磁盘空间？
**A:** 
- 安装程序：约 50-80 MB
- 便携版：约 150-200 MB
- 安装后：约 200-300 MB（包括数据缓存）

### Q: 可以在 USB 中使用吗？
**A:** 可以。使用便携版 EXE，直接复制到 USB 即可。

### Q: 如何卸载应用？
**A:** 
- **安装程序版**：从控制面板的"程序和功能"中卸载
- **便携版**：直接删除 EXE 文件即可

### Q: 遇到问题怎么办？
**A:** 
1. 检查是否已安装 Visual C++ Redistributable
2. 尝试重新安装应用
3. 确保 Windows 已更新到最新版本
4. 检查磁盘空间是否充足
